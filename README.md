# Mutual-Fund-App

## Setup virtual environment
- pip install pipenv
- pipenv shell
- pip install -r requirements.txt


## Use CLI:
`python show_mutual_fund_history_graph.py -h`

## Use GUI
`python GUI_Interface.py`
